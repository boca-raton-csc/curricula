A Case for JavaScript

If you have any programming experience, you likely have an opinion about JavaScript. Here is why we chose to teach JavaScript over other languages:

  • JavaScript can run no matter the operating system as long as a browser is installed.
  • Everything is an object.
  • Extremely flexible, can emulate syntaxes from many languages.
  • Weak typed programming languages are typically easier for beginners.
  • Constantly being updated (ES2015, ES2016).
  • Node.js is awesome.
  • HUGE community of open source libraries (NPM).
  • It's fun (opinion).

More important then the language is we will be teaching you how to think like a computer scientist. After that you should have the tools to go and learn whatever language you want.
